cursores css

1 - Cursores Padrões

Default

Código: cursor:default;

Cursor padrão, já vem como padrão pelo navegador, geralmente em formato de uma seta.

None

Código: cursor:none;

Nesta opção nenhum cursor é exibido quando o mouse estiver em cima do elemento.

2 - Cursores Comuns

Pointer

Código: cursor:pointer;

Exibe uma "mãozinha" no lugar do cursor, muito utilizada para elementos clicáveis como botões e links.

Text

Código: cursor:text;

É exibido um cursor de texto, geralmente utilizado para indicar áreas de texto editáveis.

Wait

Código: cursor:wait;

Exibe um relógio de espera, usado para indicar que o usuário deve aguardar.

Help

Código: cursor:help;

Exibe um ponto de interrogação ao cursor, geralmente indicando ajuda ou informação adicional.

Progress

Código: cursor:progress;

Roda de carregamento, muito utilizado para indicar que algo está em andamento mas é possivel fazer interação

Crosshair

Código: cursor:crosshair;

Exibe o cursor em formato de cruz, bastante utilizado para indicar precisão

3 - Cursores de Redimensionamento

N-resize

Código: cursor:n-resize;

O cursor indica que o elemento pode ser redimensionado verticalmente para o norte (para cima).

S-resize

Código: cursor:s-resize;

O cursor indica que o elemento pode ser redimensionado verticalmente para o sul (para baixo).

E-resize

Código: cursor:e-resize;

O cursor indica que o elemento pode ser redimensionado horizontalmente para o leste (direita).

W-resize

Código: cursor:w-resize;

O cursor indica que o elemento pode ser redimensionado horizontalmente para o oeste (esquerda).

Ne-resize

Código: cursor:ne-resize;

O cursor indica que o elemento pode ser redimensionado diagonalmente para o nordeste (canto superior direito).

Nw-resize

Código: cursor:nw-resize;

O cursor indica que o elemento pode ser redimensionado diagonalmente para o noroeste (canto superior esquerdo).

Se-resize

Código: cursor:se-resize;

O cursor indica que o elemento pode ser redimensionado diagonalmente para o sudeste (canto inferior direito).

Sw-resize

Código: cursor:sw-resize;

O cursor indica que o elemento pode ser redimensionado diagonalmente para o sudoeste (canto inferior esquerdo).

Ew-resize

Código: cursor:ew-resize;

O cursor indica que o elemento pode ser redimensionado horizontalmente em ambas as direções (leste e oeste).

Ns-resize

Código: cursor:ns-resize;

O cursor indica que o elemento pode ser redimensionado verticalmente em ambas as direções (norte e sul).

Nesw-resize

Código: cursor:nesw-resize;

O cursor indica que o elemento pode ser redimensionado diagonalmente, do nordeste ao sudoeste.

Nwse-resize

Código: cursor:nwse-resize;

O cursor indica que o elemento pode ser redimensionado diagonalmente, do noroeste ao sudeste.

4 - Cursores de Movimento

Move

Código: cursor:move;

Cursor padrão, já vem como padrão pelo navegador, geralmente em formato de uma seta.

None

Código: cursor:none;

Nesta opção nenhum cursor é exibido quando o mouse estiver em cima do elemento.
